# Changelog

All notable changes to Chat2Tool will be documented in this file.

## 0.1.0-alpha.0 - 2026-04-23

Initial alpha release.

- Added a local studio for turning conversations or custom briefs into reusable Alpine.js tools.
- Added a four-step wizard: source, theme, generate, review and save.
- Added conversation-first prompting that asks the model to infer title, summary, tags, and RTL direction when needed.
- Added a theme framework with multiple visual systems and color schemes.
- Added sandboxed preview, lightweight direct editing, export, and a filesystem-backed tool library.
- Added Docker support for self-hosting with persistent library storage.
